# Edgy Grid

Application React Native Expo affichant une grille 6×6 de chiffres colorés.

## Lancer l'application

```bash
cd edgy-grid
npm start
```

Ensuite, scannez le QR code avec l'application Expo Go sur votre téléphone, ou appuyez sur :
- `w` pour ouvrir dans le navigateur web
- `a` pour Android (émulateur requis)
- `i` pour iOS (macOS + Xcode requis)

## Caractéristiques

- Grille 6×6 centrée sur fond sombre (#1a1a1a)
- Chiffres de 1 à 5 générés aléatoirement
- 5 couleurs saturées distinctes :
  - 1 : Rouge (#E63946)
  - 2 : Orange (#F77F00)
  - 3 : Vert (#06D6A0)
  - 4 : Bleu (#118AB2)
  - 5 : Violet (#9D4EDD)
- Typographie nette et moderne
- Aucune interaction ni animation
